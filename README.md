# Cats vs Dogs Image Classifier

A Support Vector Machine (SVM) based image classification project that classifies images as either Cats or Dogs using the Kaggle Dogs vs Cats dataset.

## Features

- Image classification using SVM
- Image upload support
- Web interface using Streamlit
- Trained on Kaggle Dogs vs Cats dataset

## Project Structure

```
CatsDogsSVM/
│
├── app.py
├── streamlit_app.py
├── svm_cats_dogs.py
├── requirements.txt
├── templates/
├── .gitignore
└── README.md
```

## Installation

1. Clone the repository

```bash
git clone <repository-url>
cd CatsDogsSVM
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```

or

```bash
streamlit run streamlit_app.py
```
